本方是在本地的Windows电脑上安装Vuepress环境，然后配置文件、笔记等这些都是在本地编辑的，可以用Pycharm来进行Markdown编写，也可以使用Typora等软件，此外，后期为了批量执行命令，写了shell脚本，需要使用Git的CMD窗口执行sh命令，Windows环境的CMD不能使用。当本地调试并打包完成后，就是部署了，目前是将其部署在Github的新仓库中。

# Git

# 可能提示 Vue-template-compiler 未安装

npm install vue-template-compiler

# 2. 配置

# SSH

MAC配置git，解决无权限问题，具体可参考官网：https://docs.github.com/en/authentication/connecting-to-github-with-ssh/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent

# 生成密钥文件
ssh-keygen -t ed25519-sk -C "YOUR_EMAIL"

cd ~/.ssh
vim config

# 在config文件中添加如下内容
Host github.com
  AddKeysToAgent yes
  UseKeychain yes
  IdentityFile ~/.ssh/id_ed25519
  
# 在github主页ssh添加 id_ed25519.pub中的内容， 测试连接
ssh -T git@github.com
# 打印：Hi dcj00! You've successfully authenticated, but GitHub does not provide shell access.

# 
ssh-add --apple-use-keychain ~/.ssh/id_ed25519

# 就可以执行 deploy.sh 了
chmod +x deploy.sh

# mac上使用  !/bin/bash 解释器
sudo ./deploy.sh
